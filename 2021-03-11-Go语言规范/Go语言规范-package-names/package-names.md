# package-names

包命名规范：

1. 仅小写，**没有大写或下划线**。
2. 简洁简短，有代表性。
3. 不是复数，举个栗子，使用 net/url 而不是 net/urls。
4. 不要使用common,util,shared,lib等，因为信息不明确，过于宽泛。
